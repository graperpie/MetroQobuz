<h1>MetroTidal</h1>
<p>(Mostly Functioning) Tidal Music client for Android (*via binimum's hifi-api*)</p>

- replaces the YouTube API backend with the hifi-api, and converts YouTube tracks into Tidal stream URL's
- this is technically piracy, but eh piracy is cool anyway

<h2>KNOWN ISSUES</h2>
- Hi-Res Lossless may not be played and falls back to Lossless despite the label claiming it's Hi-Res Lossless
- offload won't work, but who uses that anyway
- some more idk about yet

<h2>FEATURES COMPARED TO Metrolist</h2>
- Lossless and Hi-Res Lossless (broken) streaming
- Tidal backend
- that's... pretty much it.
